# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-23

### Fixed
- Fixed mock data import paths in API routes to work correctly when copied to user projects
- Added mockData.ts to analytics API folder for proper distribution

## [1.0.0] - 2026-01-23

### Added
- Initial release
- Analytics Overview widget with metrics and 7-day trend chart
- Top Pages widget showing most visited pages
- Active Users widget with real-time data and top 3 locations
- Channel Groups widget displaying sessions by traffic source
- Dark mode support for all widgets
- Demo mode for displaying mock data
- Google Analytics Data API integration
- Service account authentication
- TypeScript support
- Comprehensive documentation
